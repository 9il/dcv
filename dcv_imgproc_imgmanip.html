    dcv.imgproc.imgmanip



    Slice!(N, V*) resize(alias interp = linear, V, size_t N, Size...)(Slice!(N, V*) slice, Size newsize) if (allSameType!Size && allSatisfy!(isIntegral, Size) && isInterpolationFunc!interp);
    Resize array using custom interpolation function.

    Primarilly implemented as image resize. 1D, 2D and 3D arrays are supported, where 3D array is treated as channeled image - each channel is interpolated as isolated 2D array (matrix).

    Interpolation function is given as a template parameter. Default interpolation function is linear. Custom interpolation function can be implemented in the 3rd party code, by following interpolation function rules in dcv.imgproc.interpolation.

    Params:
    Slice!(N, V*) slice Slice to an input array.
    Size newsize tuple that defines new shape. New dimension has to be the same as input slice in the 1D and 2D resize, where in the 3D resize newsize has to be 2D.

    Slice!(N, V*) scale(alias interp = linear, V, size_t N, Scale...)(Slice!(N, V*) slice, Scale scale) if (allSameType!Scale && allSatisfy!(isFloatingPoint, Scale) && isInterpolationFunc!interp);
    Scale array size using custom interpolation function.

    Implemented as convenience function which calls resize using scaled shape of the input slice as: 

    scaled = resize(input, input.shape*scale)


    pure Slice!(N, T*) warp(alias interp = linear, size_t N, T, V)(Slice!(N, T*) image, Slice!(3, V*) map, Slice!(N, T*) prealloc = emptySlice!(N, T));
    Pixel-wise warping of the image.

    Displace each pixel of an image by given [x, y] values.

    Params:
    interp (template parameter) Interpolation function, default linear.
    Slice!(N, T*) image Input image, which is warped. Single and multiple channel images are allowed.
    Slice!(3, V*) map Displacement map, which holds [x, y] displacement for each pixel of an image.
    Slice!(N, T*) prealloc Pre-allocated memory where resulting warped image is stored, if defined. Should be of same shape as input image, or an emptySlice, which implies newly allocated image is used.

    Returns:
    Warped input image by given map.

    pure Slice!(N, T*) remap(alias interp = linear, size_t N, T, V)(Slice!(N, T*) image, Slice!(3, V*) map, Slice!(N, T*) prealloc = emptySlice!(N, T));
    Pixel-wise remapping of the image.

    Move each pixel of an image to a given [x, y] location defined by the map. Function is similar to , except displacement of pixels is absolute, rather than relative.

    Params:
    interp (template parameter) Interpolation function, default linear.
    Slice!(N, T*) image Input image, which is remapped. Single and multiple channel images are allowed.
    Slice!(3, V*) map Target map, which holds [x, y] position for each pixel of an image.
    Slice!(N, T*) prealloc Pre-allocated memory where resulting remapped image is stored, if defined. Should be of same shape as input image, or an emptySlice, which implies newly allocated image is used.

    Returns:
    Remapped input image by given map.

    Slice!(N, V*) transformAffine(alias interp = linear, V, TransformMatrix, size_t N)(Slice!(N, V*) slice, inout TransformMatrix transform, size_t[2] outSize = [0, 0]);
    Transform an image by given affine transformation.

    Params:
    interp (template parameter) Interpolation function. Default linear.
    Slice!(N, V*) slice Slice of an image which is transformed.
    TransformMatrix transform 2D Transformation matrix (3x3). Its element type must be floating point type, and it can be defined as Slice object, dynamic or static 2D array.
    size_t[2] outSize Output image size - if transformation potentially moves parts of image out of input image bounds, output image can be sized differently to maintain information.

    note:
    Given transformation is considered to be an affine transformation. If it is not, result is undefined.

    Returns:
    Transformed image.

    Slice!(N, V*) transformPerspective(alias interp = linear, V, TransformMatrix, size_t N)(Slice!(N, V*) slice, inout TransformMatrix transform, size_t[2] outSize = [0, 0]);
    Transform an image by given perspective transformation.

    Params:
    interp (template parameter) Interpolation function. Default linear.
    Slice!(N, V*) slice Slice of an image which is transformed.
    TransformMatrix transform 2D Transformation matrix (3x3). Its element type must be floating point type, and it can be defined as Slice object, dynamic or static 2D array.
    size_t[2] outSize Output image size [width, height] - if transformation potentially moves parts of image out of input image bounds, output image can be sized differently to maintain information.

    note:
    Given transformation is considered to be an perspective transformation. If it is not, result is undefined.

    Returns:
    Transformed image.
