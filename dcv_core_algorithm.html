enum NormType: int;
Type of matrix and vector norms.

INF
Infinite norm, max(x)

L1
1-norm, sum(abs(x))

L2
Eucledian norm, sqrt(sum(x^^2))

pure nothrow ElementType!Range findMin(Range)(Range range) if (isForwardRange!Range && isNumeric!(ElementType!Range));
Find minimum value in the forward range.

Params:
Range range Forward range in which minimum value is looked for.

Returns:
Minimum value of all elements in given range.

Examples:
Find minimal value in the array. 
auto arr = [1, 2, 3];
assert(arr.findMin == 1);


pure nothrow ElementType!Range findMax(Range)(Range range) if (isForwardRange!Range && isNumeric!(ElementType!Range));
Find maximum value in the forward range.

Params:
Range range Forward range in which maximum value is looked for.

Returns:
Maximum value of all elements in given range.

Examples:
Find maximal value in the array. 
auto arr = [1, 2, 3];
assert(arr.findMax == 3);

auto normalize(Range)(Range range, NormType normType = NormType.L2) if (isForwardRange!Range && isAssignable!(real, ElementType!Range));
Normalize range using given norm type.

Performs lazy normalization by utilizing std.algorithm.iteration.map.

Params:
Range range Forward range that represent vector or matrix which is normalized.
NormType normType requested norm type for normalization.

Returns:
Returns the map result range that evaluates normalization lazily.

pure nothrow @safe auto scaled(Scalar, Range)(Range range, Scalar alpha = 1, Scalar beta = 0) if (isForwardRange!Range && isNumeric!(ElementType!Range) && isNumeric!Scalar && isAssignable!(ElementType!Range, Scalar));
Scale range values by shift and multiplication.

Performs value scaling by multiplication and shifting by following equation: 
out = alpha*range + beta


Performs lazy scale of the values by utilizing std.algorithm.iteration.map.

Params:
Range range Forward range in which values are being scaled.
Scalar alpha Multiplier value.
Scalar beta Shift value.

Returns:
Returns the map result range that evaluates value scaling lazily.

Examples:
auto arr = [0.0, 0.5, 1.0];
arr = arr.scaled(10.0, 2.0).array;
assert(arr[0] == 2.0);
assert(arr[1] == 7.0);
assert(arr[2] == 12.0);


auto ranged(Scalar, Range)(Range range, Scalar min = 0, Scalar max = 1) if (isForwardRange!Range && isNumeric!(ElementType!Range) && isNumeric!Scalar && isAssignable!(ElementType!Range, Scalar));
Scale range values to fit a given value range.

Performs lazy scale of the values by utilizing std.algorithm.iteration.map.

Params:
Range range Forward range in which values are being scaled.
Scalar min Minimal range value.
Scalar max Maximal range value.

Returns:
Returns the map result range that evaluates value scaling lazily.

Examples:
auto arr = [0.0, 0.5, 1.0].ranged(0, 10).array;
assert(arr[0] == 0.);
assert(arr[1] == 5.);
assert(arr[2] == 10.);
